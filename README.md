**CareBridge** is a high-performance clinical dashboard designed for real-time patient vitals monitoring and healthcare
providers to connect. It bridges the
gap between complex medical telemetry and actionable healthcare insights.
---

## ✨ Key Features

* **Real-time Vitals:** Live tracking of Heart Rate (BPM) and SpO2 levels.
* **Patient Context:** Full medical profiles, including a place where they can see their medical records and
  prescription, also their vitals.
* **Modern UI:** Responsive, high-contrast dashboard built with **Tailwind** and **PrimeNG**.
* **Secure Backend:** Java-based API with PostgreSQL persistence.

---

## 🛠️ The Tech Stack

### Frontend

- **Framework:** Angular
- **UI Library:** PrimeNG
- **Styles:** Tailwind CSS
- **Icons:** PrimeIcons

### Backend

- **Language:** Java 25
- **Framework:** Spring Boot
- **Database:** PostgreSQL
- **Build Tool:** Gradle

---

## 🎨 Design System

The interface is designed for high-stress medical environments, focusing on clarity and rapid data interpretation. It is
simple and clean, focusing on the comfort of the patients and the need of the healthcare providers for a simple and easy
to use application.
